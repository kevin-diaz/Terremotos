## Ejecución con Docker

1. Ejecutar el siguiente comando 
    ```
    sudo docker-compose up --build --remove-orphans
    ```
2. La aplicación estará corriendo en: 
 
    Frontend
 - http://localhost:/

    Backend:
 - http://localhost:8085/
